# lamu-leimu-button

拉姆雷姆按钮。

效果：点击拉姆回到顶部，点击蕾姆前往底部。

### 使用

```html
<body>

    <!-- 在页面最后引入 -->
    <script src="https://cdn.jsdelivr.net/gh/fz6m/lamu-leimu-button@1.3/dist/lamu-leimu.min.js"></script>

</body>
```
